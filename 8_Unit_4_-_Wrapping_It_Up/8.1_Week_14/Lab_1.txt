====== Lab 1 ======

===== Hacking Wireless Networks - TryHackMe Room =====

==== Overview ====
	Learn to attack WPA(2) networks! in this interactive walktrough TryHackMe Room.


==== Objectives ====

	Use the Aircrack-ng tool suite to capture wireless frames. Capture enough data from an Access Point to perform a dictionary attack for passphrase recovery.


==== References ====
	* Aircrack-ng wifi network security assessment tools https://www.aircrack-ng.org/doku.php?id=Main
	* Wi-Fi security - WEP, WPA, and WPA2 (Whitepaper by Guillaume Lehembre) http://repository.root-me.org/R%C3%A9seau/EN%20-%20Hacking%20wifi.pdf
